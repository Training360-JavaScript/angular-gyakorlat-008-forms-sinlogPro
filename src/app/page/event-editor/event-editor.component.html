The name of the event must be between 8 and 25 characters long.
The format of the date must be 'Day-Month-Year with leading zeros' format (17/02/2009).
The format of the time must be 12-hour format and it should be end with 'AM' or 'PM' (10:13 PM, 8:51 AM).
The format of the location must be 'address-number City Country' format (58906 Emmet Trail).
